# @borellion/beacon

Track visitor analytics in your virtual spaces with the Borellion Beacon.

## Installation

**CDN:**
```html
<script src="https://cdn.borellion.com/sdk/borellion-beacon.js"></script>
```

**NPM:**
```sh
npm install @borellion/beacon
```

## Quick Start

Add the beacon to your scene with your space's ID:

```js
import Borellion from '@borellion/beacon';

const beacon = new Borellion('YOUR_SPACE_ID');
```
